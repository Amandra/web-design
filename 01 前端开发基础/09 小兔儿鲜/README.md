# 1 项目目录结构

```shell
.
├── css # css 文件目录
│   ├── base.css
│   ├── base.less # 基础公共样式
│   ├── common.css
│   ├── common.less # 该网站多个网页相同模块的重复样式，如：头部、底部
│   ├── index.css 
│   └── index.less # 首页样式
├── favicon.ico # 网站的 ico 图片
├── images # 存放网站 固定使用 的图片素材，如：logo、样式修饰图片等。
│   ├── logo.png
│   └── sprites.png
├── index.html
├── uploads # 存放网站 非固定使用 的图片素材，如：商品图片、宣传图片等。
```
